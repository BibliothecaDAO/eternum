# Axis Install and Run

## Prerequisites

- One model provider API key (for example `ANTHROPIC_API_KEY`)
- For source builds: Node.js 22+, pnpm, Bun

## Install (Public Binary)

```bash
curl -fsSL https://github.com/bibliothecadao/eternum/releases/latest/download/install-axis.sh | bash
```

Pinned version:

```bash
curl -fsSL https://github.com/bibliothecadao/eternum/releases/latest/download/install-axis.sh | VERSION=v0.1.0 bash
```

Verify:

```bash
axis --version
```

## Quick Start (Interactive / Laptop)

```bash
# 1. Set your API key (Axis auto-creates ~/.eternum-agent/.env on first run)
echo "ANTHROPIC_API_KEY=sk-ant-..." >> ~/.eternum-agent/.env

# 2. Run — discovers worlds, opens browser for auth, starts agent
axis
```

That's it. Axis auto-initializes all runtime directories and config files on first run.

## Quick Start (Headless / VPS)

```bash
# 1. Set your API key
echo "ANTHROPIC_API_KEY=sk-ant-..." >> ~/.eternum-agent/.env

# 2. Auth — prints URL and exits (no browser needed on VPS)
axis auth my-world
#   Prints: Approve at: https://x.cartridge.gg/session?...
#   Prints: axis auth-complete my-world --redirect-url="..."

# 3. Open the URL on your laptop/phone, approve the session
#    Copy the redirect URL from your browser after approval

# 4. Complete auth on VPS
axis auth-complete my-world --redirect-url="<paste redirect URL>"

# 5. Run headless
axis run --headless --world=my-world
```

See [Headless VPS Auth](/development/axis/worlds-and-auth#headless-vps-auth-no-browser) for details.

## Run from Source

```bash
git clone https://github.com/bibliothecadao/eternum.git
cd eternum
pnpm install

# Build required workspace dependencies
pnpm --dir packages/types build
pnpm --dir packages/torii build
pnpm --dir packages/provider build
pnpm --dir packages/client build
pnpm --dir packages/game-agent build

# Run Axis
cd client/apps/onchain-agent
cp .env.example .env
pnpm dev
```

## Standalone Binary from Source

```bash
cd client/apps/onchain-agent
bun run build.ts --compile
cp axis ~/.local/bin/
axis --version
```

## Install Lifecycle

- Roll back: reinstall with `VERSION=vX.Y.Z`
- Uninstall: remove `~/.local/bin/axis`, `~/.local/share/eternum-agent`, and optionally `~/.eternum-agent`
