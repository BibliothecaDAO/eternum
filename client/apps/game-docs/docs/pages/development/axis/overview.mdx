# Axis

Autonomous onchain agent for Eternum. Explores, fights, and manages economy.

:::note[Three modes]

- **`axis run`** — autonomous tick loop (agent plays on its own)
- **`axis mcp`** — MCP server (tools called by Claude Code)
- **`axis <command>`** — one-shot CLI commands (query or act, then exit) :::
