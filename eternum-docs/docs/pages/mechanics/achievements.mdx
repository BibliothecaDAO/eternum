# Achievements & Quests

## Cartridge Achievements

Achievements are permanent records of your accomplishments across all seasons.

### Achievement Categories

🏰 **Realm Development**

- First Realm Settlement
- Building Milestones
- Resource Production Goals

⚔️ **Military**

- Army Size Thresholds
- Successful Raids
- Defense Victories

🫏 **Trading**

- Donkeys burned

🌟 **Special**

- First Hyperstructure Contribution
- Bank Control
